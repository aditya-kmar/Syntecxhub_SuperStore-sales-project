# SuperStore-sales-project
showcasing the monthly &amp; yearly trends
📊 Superstore Sales Dashboard | Power BI
📖 Project Overview
This project is an interactive Power BI Sales Dashboard built using the popular Superstore dataset. The dashboard provides meaningful insights into sales performance, profitability, customer behavior, shipping trends, and future sales forecasting.

The goal of this project is to transform raw business data into actionable insights through interactive visualizations and business intelligence techniques.

🎯 Objectives
Analyze overall business performance
Track Sales, Profit, Quantity, and Returns
Identify top-performing products and regions
Compare monthly sales and profit trends
Analyze customer purchasing behavior
Forecast future sales using historical data
Enable interactive business reporting
📊 Dashboard Features
KPI Cards (Sales, Profit, Quantity, Returns)
Monthly Sales Trend
Monthly Profit Trend
Regional Analysis
Interactive Map
Payment Mode Analysis
Customer Segment Analysis
Product Category Analysis
Sub-Category Performance
Shipping Mode Analysis
State-wise Sales Ranking
Sales Forecasting
Interactive Slicers
📈 Key Insights
💰 Total Sales: 1.6M
📈 Total Profit: 175K
📦 Quantity Sold: 22K
🔄 Total Returns: 287
🌎 West Region generated the highest sales.
👥 Consumer segment contributed the highest revenue.
🪑 Office Supplies led all product categories.
📱 Phones were the best-performing sub-category.
🚚 Standard Class was the most preferred shipping mode.
💳 COD accounted for the largest share of payments.
📍 California recorded the highest sales.
📊 Sales increased significantly during the year-end months.
🔮 Forecasting helps estimate future business performance.

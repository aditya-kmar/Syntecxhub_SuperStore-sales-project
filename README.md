# 🚀 Superstore Sales Dashboard

## 📖 Project Overview

This project is an interactive Power BI dashboard that provides comprehensive insights into sales performance, profitability, customer behavior, and forecasting.

## 📊 Dashboard Features

- KPI Cards
- Sales Analysis
- Profit Analysis
- Regional Analysis
- Forecasting

## 📈 Key Insights

- Total Sales: **1.6M**
- Total Profit: **175K**
- West Region generated the highest sales.
- Consumer Segment contributed the highest revenue.

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Excel
